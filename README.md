# Mohana Medisetty — Professional Portfolio

A senior-level professional portfolio site showcasing expertise, experience, recognition, and impact. Built for job applications and professional visibility.

## Contents

- **About** — Professional summary and background
- **Expertise** — AI/ML systems, backend & platforms, DevOps & cloud, product & delivery
- **Experience** — Detailed roles with impact metrics (Wand AI, CXAI, Salk, Experian, DevC, Smarter.Codes)
- **Education** — University of San Francisco (MS Data Science), JNTU (BS Computer Science)
- **Recognition & Impact** — HerTech Hackathon winner, Facebook DevC recognition, Grace Hopper, Wand × Presight press
- **Featured Work** — AI Workplace Assistant demo, Agent Systems, Metadata Hub, Biomedical ML pipeline
- **Community & Leadership** — Code First Girls, WISE, Developer Circles
- **Contact** — Email, LinkedIn, GitHub, location

## How to view

1. **Local:** Open `index.html` in a browser, or run a simple server:
   ```bash
   npx serve .
   ```
   Then open http://localhost:3000 (or the port shown).

2. **Deploy:** Upload `index.html` and `styles.css` to any static host (GitHub Pages, Netlify, Vercel, etc.).

## Files

- `index.html` — Single-page structure and content
- `styles.css` — Layout, typography, responsive and print styles
- `README.md` — This file

## Customization

- **Contact:** Update email and links in the Contact section and hero.
- **Content:** Edit section text in `index.html` to keep experience, awards, and links current.
- **Design:** Adjust CSS variables in `:root` in `styles.css` (colors, spacing, fonts).
